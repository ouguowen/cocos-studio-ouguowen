# Milestone Burndown Rules

Use this file when tracking whether a milestone is actually converging or only appearing busy.

## Burndown law

- Count remaining scope, not just completed tasks.
- Separate blocker work from ordinary feature work.
- If scope grows faster than closure, call that out directly.

## Required views

- total committed scope
- completed scope
- blocked scope
- added scope
- removed scope
- burn risk

## Template

```md
# Milestone Burndown

## Milestone

## Owner

## Date

## Total Committed Scope

## Completed Scope

## Blocked Scope

## Added Scope Since Start

## Removed Scope Since Start

## Main Burn Risks

## Probability Of Hitting Date

## Escalation Decision
```
