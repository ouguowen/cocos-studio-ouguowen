# Outsourcing Collaboration Rules

Use this file when external artists, animators, programmers, QA vendors, or content vendors are involved.

## Outsourcing law

- No external contributor should receive vague ownership.
- No outsourced deliverable should be accepted without format, review, and integration rules.
- No vendor should become the hidden owner of core architecture.

## Required rules

- delivery owner
- review owner
- file format
- naming rules
- handoff checklist
- revision rounds
- acceptance criteria
- integration owner
