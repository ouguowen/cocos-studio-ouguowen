# Cost Budget Model

Use this file when a game, feature, event, or release needs explicit cost discipline.

## Cost law

- If runtime cost has no owner, it will drift.
- If asset cost is not budgeted early, late cuts will be destructive.
- If external service cost is invisible, live operations will surprise the team.

## Cost buckets

- art production
- audio production
- engineering time
- QA time
- backend or platform services
- ad acquisition or campaign spend
- localization
- outsourcing
- live operations
