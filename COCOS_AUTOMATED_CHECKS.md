# Cocos Automated Checks

Use this file as the minimum automated review pack for Skill-level Cocos governance changes.

## Checks

- repository path-scope check
- blocked-path pattern check
- required-doc existence check
- Agent count and registry integrity check
- command registry linkage check
- quality-gate linkage check
- audit-report freshness check

## Minimum outputs

- missing files
- missing links
- blocked-path violations
- Agent count result
- PASS / FAIL summary
