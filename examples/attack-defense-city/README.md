# Attack Defense City Example

This example is a tiny level-config seed for a modern city attack-defense game.

## Purpose

Use it to test the level config validation and export pipeline.

## Run validation

From the repository root:

```bash
npm run validate:example
```

## Export runtime JSON

```bash
npm run export:example
```

## MVP design

- One city map
- One player spawn point
- Two enemy spawn points
- One enemy family
- Two waves
- One required objective: kill all enemies
- One reward
