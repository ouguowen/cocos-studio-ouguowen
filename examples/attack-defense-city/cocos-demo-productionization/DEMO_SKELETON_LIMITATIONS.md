# Demo Skeleton Limitations

This document prevents overclaiming.

## What exists now

The repository currently provides:

- AI Game Studio skill rules
- modern city attack-defense example pack
- level config data
- validation scripts
- Cocos integration guides
- Cocos reference stubs
- setup guides
- demo skeleton documentation
- QA gates and logs

## What does not exist yet

The repository does not yet provide:

- a full Cocos Creator project
- real `.scene` files created by Cocos Creator
- real `.prefab` files created by Cocos Creator
- verified runtime execution inside the Cocos editor
- final art or animation
- production-ready enemy behavior
- production-ready UI

## Correct wording

Use this wording:

```text
Cocos demo skeleton
Cocos reference stub
Cocos setup guide
example pack
```

Do not use this wording yet:

```text
complete runnable Cocos game
finished Cocos demo
production-ready project
```

## Upgrade condition

Only upgrade the wording after:

1. A real Cocos Creator project is created.
2. Scripts compile in the editor.
3. Scene and prefab assets exist.
4. Inspector bindings are verified.
5. First run validation log is filled.
6. QA gate passes.
