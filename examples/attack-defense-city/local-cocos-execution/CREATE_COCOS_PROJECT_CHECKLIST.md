# Create Cocos Project Checklist

Use this checklist when creating or opening the local Cocos Creator 3.8.8 project.

## Project setup

- [ ] Cocos Creator version is 3.8.8
- [ ] project opens without editor errors
- [ ] project is a 2D project or has a 2D scene setup
- [ ] project folder is known and easy to find

## Folder setup

- [ ] `assets/scenes/` exists
- [ ] `assets/scripts/attack-defense-city/` exists
- [ ] `assets/prefabs/enemies/` exists
- [ ] `assets/resources/config/` exists

## Script setup

- [ ] reference stubs copied or adapted
- [ ] scripts compile
- [ ] `AttackDefenseCityBootstrap` is available in Inspector
- [ ] `MapPointComponent` is available in Inspector

## Config setup

- [ ] `generated-level-config.json` copied into `assets/resources/config/`
- [ ] config source was generated from repository command
- [ ] config was not manually edited inside Cocos

## Scene setup

- [ ] `scene_city_battle.scene` created
- [ ] required nodes created
- [ ] point IDs assigned
- [ ] prefabs assigned

## First run setup

- [ ] scene can be played in editor
- [ ] console output is checked
- [ ] errors are copied into proof report
