# Cocos Studio Ouguowen

Senior production-system skill for Cocos Creator 3.x game development.

## What this repository contains

- `SKILL.md`: skill entry and routing rules
- `ROLES.md`: role responsibilities
- `ROLE_STAGE_MATRIX.md`: role-by-stage execution map
- `STAGES.md`: stage system and gates
- `OWNERSHIP.md`: asset ownership rules
- `COCOS_RULES.md`: Cocos Creator 3.x anti-chaos rules
- `PROJECT_STRUCTURE.md`: default project blueprint
- `QUALITY_GATES.md`: quality and release gates
- `CHECKLISTS.md`: stage and release checklists
- `WORKFLOWS.md`: execution workflows
- `EXAMPLES.md`: applied examples
- `TEMPLATES.md`: project document templates

## Purpose

This skill is designed to help a solo developer or small team run a Cocos Creator 3.x game project with senior-level discipline instead of ad hoc production.
